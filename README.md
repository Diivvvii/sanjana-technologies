# Sanjana Technologies

Sanjana Technologies is a modern and responsive business website designed to showcase the company's services, expertise, and contact information. The website is built using HTML, CSS, and JavaScript with a clean, user-friendly interface that works seamlessly across desktop, tablet, and mobile devices.

## Live Demo

https://diivvvii.github.io/sanjana-technologies/

## Features

- Responsive design
- Modern and clean user interface
- Hero section with call-to-action
- About section
- Services section
- Why Choose Us section
- Contact section
- Smooth scrolling navigation
- Mobile-friendly navigation menu
- Fast loading and lightweight

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/Diivvvii/sanjana-technologies.git
```

Navigate to the project directory:

```bash
cd sanjana-technologies
```

Open `index.html` in your preferred web browser.
